# AI Trainer - Teachable Clone (Web)

Este é um clone funcional do **Teachable Machine**, desenvolvido puramente em HTML5, CSS3 (Tailwind) e JavaScript, utilizando o poder do **TensorFlow.js**.

## 🚀 Funcionalidades
- **Transfer Learning**: Utiliza o modelo MobileNet v1 para extrair características das imagens.
- **KNN Classifier**: Treina um classificador de vizinhos mais próximos em tempo real no navegador.
- **Interface Intuitiva**: Customize os nomes das classes e segure para gravar amostras.
- **Predição Instantânea**: Visualização de confiança via barras de progresso animadas.

## 🛠️ Tecnologias
- [TensorFlow.js](https://www.tensorflow.org/js)
- [MobileNet](https://github.com/tensorflow/tfjs-models/tree/master/mobilenet)
- [Tailwind CSS](https://tailwindcss.com/)

## 📦 Como rodar localmente
1. Clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
   - *Nota: O acesso à câmera requer HTTPS ou localhost.*

## 🌐 Deploy no GitHub Pages
Este projeto foi otimizado para o GitHub Pages. Basta subir os arquivos para o repositório e ativar o Pages nas configurações (Settings > Pages).
